Cribl has released Cribl-Privacy 1.5, an upgraded model for detecting sensitive data in telemetry streams that catches significantly more protected information while maintaining the same computational footprint as version 1.0. The model is purpose-built for production logs and machine-generated records, improving recall without requiring additional compute resources.
